This application ("the App") provides automated responses to messages received through Instagram.

1. Information We Collect

The App may receive the following information when users interact with the connected Instagram account:

  • Instagram user ID
  • Instagram username
  • Message content sent to the Instagram account
  • Public profile information shared within messages

2. How We Use Information

The information received is used solely to:

  • Provide automated responses to messages

The App does not permanently store personal data. Message data is processed in real-time and is not retained beyond what is necessary to provide automated responses.

4. Data Sharing

No personal data is shared with third parties. All data is processed in accordance with Meta's Platform Terms and Instagram Platform Policies.
